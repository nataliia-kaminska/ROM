@startuml ResearchOpportunityMatcher_ApplicationStateDiagram
title Research Opportunity Matcher - Application/Kanban State Diagram

skinparam shadowing false
skinparam state {
  BackgroundColor #FFFDF8
  BorderColor #6B7D73
  ArrowColor #40544B
}

[*] --> Browsing : opportunity visible in catalog

Browsing --> Saved : Save
Browsing --> Ignored : Ignore

Saved --> Planned : start application planning
Saved --> Ignored : not relevant
Saved --> Browsing : remove from board

Planned --> Applied : submit application
Planned --> Saved : pause planning
Planned --> Ignored : abandon

Applied --> Accepted : positive decision
Applied --> Rejected : negative decision
Applied --> Planned : revise or resubmit

Accepted --> [*] : opportunity completed
Rejected --> [*] : opportunity closed
Ignored --> [*] : hidden from active workflow

state Saved {
  [*] --> ReminderOptional
  ReminderOptional --> ReminderScheduled : create reminder
  ReminderScheduled --> ReminderCompleted : reminder completed
}

state Planned {
  [*] --> AssistantAvailable
  AssistantAvailable --> ChecklistGenerated : generate advisor memo
  ChecklistGenerated --> ReminderScheduled : create deadline reminder
}

note right of Browsing
The opportunity can come from Grants.gov,
EU Funding and Tenders, Erasmus, NAWA,
OpenAlex-related enrichment, or other connectors.
end note

note right of Planned
Apply Assistant uses retrieved context,
parsed requirements, profile evidence,
and optional DuckDuckGo web research.
end note

note bottom of Ignored
Ignored opportunities are excluded from
personalized active match lists unless requested.
end note

@enduml
